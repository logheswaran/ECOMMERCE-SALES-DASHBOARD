# E-Commerce Sales Dashboard (Power BI)

## Project Overview
This project presents a single-page interactive E-Commerce Sales Dashboard built using Power BI.
The dashboard provides insights into sales performance, product categories, payment methods, and customer behavior.

## Tools Used
- Power BI (Dashboard & DAX)
- Python (Data Cleaning & EDA)
- Pandas, Matplotlib
- Kaggle Dataset

## Dashboard Features
- KPI cards for Total Sales, Orders, Quantity, and Average Order Value
- Monthly sales trend analysis
- Revenue breakdown by product category
- Payment method distribution
- Order status overview
- Top customers by revenue
- Dynamic filtering by Month and Product Category

## Key Insights
- Monthly sales show fluctuations with a decline toward year-end
- Electronics and Clothing are top revenue-generating categories
- Digital payment methods dominate transactions
- Majority of orders are successfully completed
- A small group of customers contributes significantly to total revenue
